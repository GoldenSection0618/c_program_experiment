# 计费管理系统编译与运行说明

## 1. 环境要求
- 操作系统：Linux / macOS / Windows（MinGW）
- 编译器：`gcc`（支持 C11）
- 构建工具：`make`

## 2. 编译
```bash
make
```
- 编译成功后会在项目根目录生成可执行文件：`billing_system`

## 3. 运行
```bash
./billing_system
```

## 4. 清理编译产物
```bash
make clean
```

## 5. 验收测试输入（可选）
程序运行后，依次输入：
`1 2 3 4 5 6 7 8 9 1w ttt 0`
